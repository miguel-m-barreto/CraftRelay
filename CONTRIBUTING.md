# Contributing

Use English for source identifiers and tests. Keep changes bounded to the active sprint. Run the commands in `README.md`; do not silently install tools. New process, durability, identity, state, query, Bridge, or trust-boundary decisions require an ADR. `unsafe` requires an ADR. Never commit generated audit bundles, secrets, `target`, or Java build output. Do not claim a durability guarantee from Sprint 0 skeletons.
